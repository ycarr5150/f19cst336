What is a virus?

A virus is a type of malware designed to replicate itself by modifying and inserting it's own code into other programs in order to spread from host to host. It is one of the most common types of malware.

The virus requires someone to knowingly or unknowingly spread the infection. This can be form opening an attachment from an email, downloading an executable file, or visiting an unsecure website.



Types of viruses include:
  • File Infectors
  • Marco Viruses
  • Overwrite Viruses
  • Polymorphic Viruses
  • Resident Viruses
  • Rootkit Viruses
  • System or Boot-Record Infectors
How to prevent:
  • Install current antivirus
  • Run daily scans
  • Regularly patch
  • Don't click on suspicious links
